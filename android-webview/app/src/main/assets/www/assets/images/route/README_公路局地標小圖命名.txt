公路局地圖頁的「到站蹦出放大」會自動尋找以下檔名：

landmark-01.webp / landmark-01.png / landmark-01.jpg
landmark-02.webp / landmark-02.png / landmark-02.jpg
...
landmark-06.webp / landmark-06.png / landmark-06.jpg

也支援：station-01、route-01、spot-01、01 這幾種前綴。

建議放 6 張透明 PNG 或 WebP，小圖本身不用太大，約 600～900px 寬即可。
當光點走到對應站點時，小圖會在該地標附近彈出放大。
