# Dear ANG v20｜開場 11 張個人照倒數版

這版依照你的要求修改開場：

## 新流程

按「進入有你的時光」後：

1. 抽出 11 張個人照。
2. 顯示 10 秒倒數。
3. 倒數到 0 秒時開始播放主題曲。
4. 進入主街景世界。

## 照片來源

優先讀 Google Drive：

`03_他的家`

底下所有子資料夾都會讀。如果暫時還沒讀到，會先顯示等待格；雲端資料回來後會自動補上。

## 音樂

主題曲仍固定使用 GitHub：

`assets/audio/dear-ang-theme.mp3`

為了避免 iPhone 擋音樂，按下開始按鈕時會先靜音解鎖音訊，倒數 0 秒時正式播放。

## 保留

- 開場歌曲資訊
- 照片滿版輪轉卡片
- KTV 舞台與圖片遙控器
- 時光膠囊
- 歌詞提前 20 秒
- Google Drive 子資料夾遞迴
