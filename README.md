# Dear ANG v29｜小世界中轉＋天氣同步版

## 這版完成

- 原本開場頁不動。
- 原本 opening-countdown.mp4 開場影片不動。
- 開場影片結束後，不直接進主街道，先進「木框小世界」。
- 點「點進這個世界」後，畫面放大淡出，才進入原本富宜路街道主頁。
- 小世界與主街道共用同一組天氣狀態。
- 右上角天氣按鈕可循環切換天氣。
- 小世界下方可直接點天氣：晴天、炎熱、多雲、陰天、雨天、暴雨、霧、霾、夜晚、夜雨。
