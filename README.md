## 新流程

按「進入有你的時光」後：

1. 顯示倒數影片。
2. 影片裡本身已有秒數，所以網頁不再額外疊數字。
3. 影片播完後視為 0 秒。
4. 0 秒正式播放主題曲。
5. 進入主街景世界。
